FlexGridify 2024

FlexGridify is a responsive flex layout library that dynamically adjusts column count and spacing for flexible, adaptive designs. Use the resize button in the bottom left corner to drive.
Documentation
Open DevTool (Press F12) and go to Console, now resize the FlexGridify, if `logQuery` is set to `true` it will log out different breakpoints.
Recieves `px` or `em` units. It will affect `gap`, `marginTop` and `marginBottom`.

Sets a `gap` between elements within FlexGridify.
Sets a `margin-top` for each element within FlexGridify.
Sets a `margin-bottom` for each element within FlexGridify.
Sets the default amount of columns in case if no `breakpointColumns` matches the current window/element size or if 'responsive' is set to `false`.
Disables `breakpointColumns` (media queries).
Sets a media query and corresponding amount of columns for the FlexGridify. Uses `min-width` query. Can be affected by the `breakpointSelector` and `relative` parameters. For no mathing query `defaultColumnAmount` will be used.
Automatically updates height if it's changed within the FlexGridify. (If text of the child will wrap or if child will change its height it will result an overall overflow of FlexGridify and can lead to unexpected behaviour when `dynamicHeight` is set to `false`)
Sets an element that will be observed for the media queries (breakpointColumns). By default is set to the main screen ("window") can be set to any element.
Calls a callback function when media query is triggered.
